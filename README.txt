EL USO QUE SE LE DE A LA IA ES DE ESTRICTA RESPONSABILIDAD DEL USUARIO

------------------Pasos para ejecutar el programa:------------------
1. Instalar python desde la pagina oficial
https://www.python.org/downloads/
Pueden seguir un tutorial de youtube como este:
https://www.youtube.com/watch?v=xd_0RN2SyfI (Desde el minuto 0 hasta el 2:22)

2.Descargar el software de voiceVox y recordar donde se hizo la instalacion
https://voicevox.hiroshiba.jp/

3.Copiar la carpeta VOICEVOX que se genero en la instalacion del paso 2 a la
carpeta raiz del proyecto.

4.Doble click en el archivo runVoiceVox.bat, el cual es el encargado de 
generar las voces del chat AI

(NO SE DEBE CERRAR CUANDO ESTES OCUPANDO EL PROGRAMA, YA QUE DARA ERROR
AL ENTREGAR RESPUESTAS Y REPRODUCIR LOS AUDIOS)

5.Doble click en el archivo ejecutarServer.bat, el cual es el encargado de
levantar un servidor local para poder usar el programa.

------------------Consideraciones adicionales------------------
-Si quieres enviar mensajes mediante el reconocimiento de tu voz, debes
ocupar el navegador chrome o edge
